# Forgejo-runner-windows-built

> This repository is the successor to [forgejo-runner-windows](https://github.com/Crown0815/forgejo-runner-windows).

This repository contains build scripts for creating [Forgejo-runner](https://code.forgejo.org/forgejo/runner) windows binaries.

The built binaries are available on the [release page](https://github.com/Crown0815/forgejo-runner-windows-built/releases).

The original code for the [Forgejo-runner](https://code.forgejo.org/forgejo/runner) is available [here](https://code.forgejo.org/forgejo/runner).
